# NickKaramoff/template

Template for a simple repo structure. Includes a LICENSE, a basic .gitignore,
and an EditorConfig.

You can use [cookiecutter][cookiecutter] to use this template:

```sh
cookiecutter https://github.com/NickKaramoff/template
```

Alternatively, you can click "Use this template" and replace the values by
yourself

**PRO TIP:** Fork this repository, replace the values with your own and use it
as a private template for your future repositories!

[cookiecutter]: https://github.com/cookiecutter/cookiecutter
